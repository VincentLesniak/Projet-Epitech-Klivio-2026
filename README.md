# Projet Epitech : Intégration Web - Klivio

## À propos du projet

Ce projet consiste à construire la page d'accueil de **Klivio**, une plateforme proposant des formations en ligne, à partir d'une maquette FIGMA. L'objectif principal est de réaliser une intégration web propre, responsive et accessible , puis de la déployer publiquement sous forme de site statique.

Le projet se divise en deux phases distinctes :

* Une intégration HTML/CSS classique fidèle au design fourni.


* Une seconde version de la page recréée à l'aide du framework Tailwind CSS, avec une liberté artistique totale sur le design visuel.



## Contenu intégré

La page d'accueil comprend plusieurs sections clés :

* **Header :** Logo, menu de navigation et boutons d'inscription/connexion.

* **Section Héro :** Titre principal, slogan et bouton d'appel à l'action (CTA).

* **Contenu principal :** Mise en avant des formations populaires, avantages et témoignages.

* **Footer :** Informations de contact, liens légaux et réseaux sociaux.


## Technologies et Outils

* **HTML5 :** Utilisation stricte de balises sémantiques et de titres hiérarchisés.

* **CSS3 :** Mise en page fluide via Flexbox ou Grid, avec un fichier externe obligatoire et aucun CSS inline.

* **Tailwind CSS :** Utilisation des classes utilitaires pour la seconde version.

* **Ressources externes :** Typographie "Montserrat" via Google Fonts et icônes via Font Awesome.

* **Hébergement :** Déploiement en ligne géré par GitHub Pages depuis la branche `main`.



## Critères de Qualité et d'Accessibilité

Ce projet respecte des normes strictes de développement front-end :

* Rendu 100% responsive, adapté aux écrans d'ordinateur et de mobile.

* Validation du code par le W3C.

* Application des bonnes pratiques SEO (balises meta, structure).

* Accessibilité garantie par des attributs `alt` sur les images, des contrastes corrects et une navigation au clavier fonctionnelle.



## Déploiement

* Le site statique est hébergé en ligne et se met à jour automatiquement à chaque `git push` sur la branche principale.

* Les deux versions du projet (HTML/CSS et Tailwind) sont consultables publiquement via des liens directs en haut de page.
